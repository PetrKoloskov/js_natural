# js_natural
<p>Для разбиения строки на слова и пойска базовой формы слов использовалась библиотека https://github.com/NaturalNode/natural.</p>
<p>Поиск базовой формы в русском тексте и английском отличается используемым методом, поэтому реализовал выбор языка(мало на что влияет, но иногда не распознает русские тексты при использовании стеммера для английского языка и наоборот)
На данный момент корректно работает поиск и подсветка одинаковых слов и более-менее подсветка однокоренных английских слов.</p>
<p>В папке ./dist находится html-страница и собранный js-файл.</p>
